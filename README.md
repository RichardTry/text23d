# Text to 3D

## Install project

```
git clone https://github.com/RichardTry/text-to-3d.git

cd text-to-3d
```

## Create environment

```
python -m venv venv_text_to_3d
source venv_text_to_3d/bin/activate
```

## Install requirements

```
pip install -r requirements.txt
```

## Run the project

```
python3 main.py
```
